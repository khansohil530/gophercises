# Gophercises

---

Gophercises are a bunch of exercises solved using Golang. 
The main motive is to learn Go by doing. 
You can check out the problems [here](https://courses.calhoun.io/courses/cor_gophercises).
This repo will contain the implementation of each of them while giving these problems a go.

1. [Quiz](quiz): You'll learn about reading and parsing csv file, taking user input from `Stdin`, goroutines and channels.
2. [URL Shortner](url-shortner): You'll learn about http request Handler, redirects, parsing yaml and creating package.
3. [Choose your own Adventure](cyoa): You'll learn about html templating, handling json data and http handlers.
